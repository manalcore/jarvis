## 🚀 DEPLOY FREE IN 5 MINUTES

### Option 1 — Vercel (Recommended)
```bash
# 1. Push to GitHub
git init && git add . && git commit -m "Mission Jarvis Agentic OS"
git push origin main

# 2. Import at vercel.com → New Project → Import your repo
# 3. Add environment variable: ANTHROPIC_API_KEY = sk-ant-...
# 4. Deploy → Done ✅
```

### Option 2 — Direct API Key (Personal use only)
Open `public/index.html`, find line:
```javascript
var DIRECT_KEY = ''; // e.g. 'sk-ant-api03-...'
```
Change to your key. Open the file in Chrome/Edge. Done.

### Option 3 — Netlify
```bash
npm install netlify-cli -g
netlify deploy --prod --dir public
# Set ANTHROPIC_API_KEY in Netlify dashboard
```

## 💡 How It Works

When you say "Avengers Assemble":

1. **Wave 1** — ORACLE (research) + HERMES (web intel) run **in parallel**
2. **Wave 2** — MERCURY (content) + ATHENA (strategy) + NEXUS (memory) run **in parallel** using Wave 1 results
3. **Wave 3** — JARVIS synthesizes everything into a master briefing
4. **Wave 4** — ECHO converts the briefing to natural speech
5. **Voice delivery** — Jarvis speaks the full report to you

Total time: ~15-30 seconds for a complete multi-agent intelligence briefing.
